# Credit Risk Prediction with Explainability

**TC2038 - Midterm 1**

Team: Santiago Arista Viramontes, Diego Vergara Hernández, José Leobardo Navarro Márquez

## Overview

Explainable credit risk scoring system using SHAP on Home Credit Default Risk dataset.

Implemented features:
- SHAP explanations (M1)
- Robustness testing (M2)
- Fairness metrics (M3)
- Governance framework (M4)

## Usage

```bash
pip install -r requirements.txt
python src/train.py
```

Results saved to `experiments/`

**That's it!** Everything is automated in one script.
